🚀 Step-by-Step: Deploy Your Portfolio on GitHub Pages

✅ Step 1: Create a GitHub Repo

  1. Go to https://github.com/new

  2. Repo name: chayesh-portfolio

  3. Keep it Public

  4. Click Create repository

✅ Step 2: Upload Your Files

  1. On your computer, create a folder called chayesh-portfolio

  2. Inside it, create a file named index.html

  3. Copy-paste the code from the Canvas (or download it from here if you want)

  4. Drag and drop this file into the GitHub repo:

    • Go to your new repo → Click Add file → Upload files

    • Upload the index.html and click Commit changes

✅ Step 3: Enable GitHub Pages

  1. In your repo, go to Settings

  2. Scroll to Pages on the left sidebar

  3. Under Source, choose:

    • Branch: main

    • Folder: /root

  4. Click Save

  5. After a few seconds, GitHub will give you a live link like:

    arduino
    https://yourusername.github.io/chayesh-portfolio/
